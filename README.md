# System Requirements

- Visual Code
- Flutter Extension
- ADV Manager Android

# Libaries

- [Sqflite](https://pub.dev/packages/sqflite)
- [Path_Provider](https://pub.dev/packages/path_provider)
- [Intl](https://pub.dev/packages?q=intl)

# Features

- Add, update and delete task
- Choose date to complete
- Choose Priority
- Mark as done
